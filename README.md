# ConGa - 변호사 계약서 품질 검증 서비스

## 서비스 개요
변호사-의뢰인 계약 분쟁을 줄이기 위한 **비영리 프로젝트** (100% 무료)

---

## MVP 기능

### 1. 계약 전 체크리스트 (3단계 모달)
| 단계 | 내용 |
|------|------|
| Step 1 | 여러 변호사 상담 여부, 담당 변호사 직접 상담 여부 확인 + 변호사 등록 확인 링크 |
| Step 2 | "이런 변호사는 피하세요" 경고 (현금할인, 착수금만 많음 등) |
| Step 3 | 메인 분석 화면 |

---

### 2. 계약서 분석 엔진

#### 필수 조항 체크 (19개)
- 담당변호사, 변호사등록번호, 위임범위
- 착수금, 환불조건, 성공보수, 성공기준
- 추가비용, 예치금액, 출장비기준
- 통지의무, 자료보관기간, 비밀유지
- 조정화해권한, 시간당요율, 변호사책임
- 강제집행범위, 승소범위정의, 시간차지방식

#### 위험 패턴 감지 (24개)
| 위험도 | 예시 |
|--------|------|
| 매우높음 | 72시간 환불불가 조항, 현금할인 제안 |
| 높음 | 팀제운영, 환불불가, 추가비용 애매, 별도협의, 적정금액, 경제적이익 모호 |
| 중간 | 담당변경가능, 성공기준모호, 포괄위임, 관례따름, 소송물가액기준 |
| 낮음 | 소통불명확, 필요시조치 |

#### 구체성 검증
- 모호한 표현 감지: "적정", "상당한", "합리적", "별도 협의", "통상적인" 등
- 구체적 표현 확인: 금액(원, 만원), 비율(%), 기간 등

---

### 3. 분석 결과
- **종합 점수**: 0~100점
- **위험 조항**: 개수 + 상세 설명 + 대응 방법
- **필수 조항**: 있음/없음 체크
- **구체성 문제**: 모호한 조항 목록

---

### 4. Double Check 질문
계약서에 있어도 직접 확인해야 할 사항:
- 가압류/가처분 포함 여부
- 시간당 차지 금액
- 화해 시 성과보수 여부
- 일부 승소 비율 계산
- 해지 시 환불 계산

---

### 5. 다운로드 기능
- 상세 리포트 (TXT)
- 변호사 개선 요청서 (TXT)

---

### 6. 개인정보 보호
- 원본 파일 미저장 (메모리에서만 처리)
- **분석 결과 5분 후 자동 삭제**
- 즉시 삭제 버튼
- 카운트다운 표시

---

### 7. 지원 형식
- PDF 파일
- JPG, PNG 이미지 (OCR 예정)
- 텍스트 직접 입력

---

## 실행 방법

### 1. 필요 라이브러리 설치
```bash
pip install -r requirements.txt
```

### 2. 실행
```bash
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 자동 실행

## 면책 조항

1. 본 서비스는 **참고 자료**이며 **법적 자문이 아닙니다**.
2. 최종 계약 결정은 반드시 변호사와 직접 상담 후 하시기 바랍니다.
3. 본 서비스의 분석 결과로 인한 어떠한 손해에 대해서도 책임지지 않습니다.
